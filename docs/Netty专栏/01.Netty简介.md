# Netty简介

<p align="center">
  <img src="https://s1.ax1x.com/2020/07/23/Uq1f4e.png" alt="Uq1f4e.png" border="0" />
</p>

Netty是一个NIO客户端服务器框架，可快速轻松地开发网络应用程序，例如协议服务器和客户端。它极大地简化和简化了网络编程，例如TCP和UDP套接字服务器。
